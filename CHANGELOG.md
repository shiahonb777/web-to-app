# 更新日志（Changelog）

所有显著的项目更改都将记录在此文件中。

## [1.0.6] - 2025-12-09

### ✨ 新增功能
- 🤖 AI 字幕生成 - 音频自动生成 LRC 歌词，时间轴精准
- 🎨 主题系统 - 多种精美主题，支持动画效果定制

### 🐛 修复
- 修复 WebView 兼容性问题
- 优化内存使用

### 📝 更改
- 更新依赖包版本

## [1.0.5] - 2025-11-15

### ✨ 新增功能
- 💻 HTML 转 App - HTML/CSS/JS 项目转换为独立应用
- 🤖 AI 编程助手 - AI 辅助生成 HTML 代码，实时预览

### 🐛 修复
- 修复应用启动崩溃问题

## [1.0.4] - 2025-10-20

### ✨ 新增功能
- 🎬 媒体转 App - 图片/视频一键转换为全屏展示应用
- 📦 一键出包 - 无需 Android Studio，直接生成可安装 APK

### 📝 更改
- 优化界面设计
- 提升应用性能

## [1.0.3] - 2025-09-10

### ✨ 新增功能
- 🌐 网站转 App - 输入任意网址，生成独立 WebView 应用
- 🛡️ 广告拦截 - 内置广告拦截引擎，自动过滤网页广告

### 🐛 修复
- 修复网络连接问题

## [1.0.0] - 2025-08-01

### ✨ 初始发布
- 🌐 网站转 App 基础功能
- 📱 应用管理
- 🔧 应用修改器

## 版本说明

- **🟢 稳定版** - 已充分测试，可用于生产环境
- **🟡 测试版** - 包含新功能，可能存在已知问题
- **🔴 开发版** - 开发中版本，不推荐使用

## 如何升级

1. 从 [Release 页面](https://github.com/yingcaihuang/web-to-app/releases) 下载最新 APK
2. 卸载旧版本应用
3. 安装新 APK 文件
4. 根据提示进行必要的权限授予
